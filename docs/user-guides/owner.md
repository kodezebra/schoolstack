# School Owner Guide

**Role:** `owner` | **Version:** 1.0

---

## Welcome

As the School Owner, you have full access to KidzKave. You can manage your entire school operation, delegate responsibilities to staff, and monitor all activities. This guide walks you through everything you need to know.

## Managing Users

### Viewing Users

Navigate to **Users** to see all team members:

1. Click **Users** in the sidebar
2. View user table with name, email, role, join date
3. Use search to find specific users

### Creating a New User

1. Click **Add User**
2. Fill in details:
   - **Name**: Full name
   - **Email**: Work email (must be unique)
   - **Password**: Temporary password (min 8 characters)
   - **Role**: Select appropriate role
3. Click **Create User**
4. Share credentials securely with the new user

### Assigning Roles

| Role | Use For |
|------|---------|
| **Admin** | Operations manager, head teacher |
| **Editor** | Content managers, regular staff |
| **Viewer** | Board members, observers |

**Best Practice:** Grant minimum permissions needed. Don't give admin access unless necessary.

### Deleting a User

1. Find user in the table
2. Click the **⋮** menu
3. Select **Delete**
4. Confirm deletion

> ⚠️ **Important:** You cannot delete yourself or the last admin owner.

## Academic Setup

Before adding students, set up your academic structure:

### 1. Create Academic Year

1. Navigate to **School** → **Academic Years**
2. Click **Add Year**
3. Enter:
   - Year name (e.g., "2024-2025")
   - Start date
   - End date
4. Mark as **Active** if this is the current year

### 2. Create Terms

1. Go to **School** → **Terms**
2. Click **Add Term**
3. Select academic year
4. Enter:
   - Term name (e.g., "Term 1")
   - Start date
   - End date

### 3. Create Levels (Grades)

1. Navigate to **School** → **Levels**
2. Click **Add Level**
3. Enter:
   - Level name (e.g., "Primary 1")
   - Code (e.g., "P1")
   - Category (Nursery/Primary/Secondary)

### 4. Create Subjects

1. Go to **School** → **Subjects**
2. Add common subjects or create custom ones
3. Subjects auto-assign to all levels

## Managing Staff

### Adding Staff Members

1. Navigate to **School** → **Staff**
2. Click **Add Staff**
3. Fill in details:
   - First name, Last name
   - Gender
   - Email, Phone
   - Role (Teacher, Administrator, etc.)
   - Assigned Level (optional)
4. Click **Save**

### Assigning Subjects to Teachers

1. Go to **School** → **Settings**
2. Find **Level Subjects** section
3. Select a level
4. Assign teachers to subjects

---

## Managing Students

### Adding Students

1. Navigate to **School** → **Students**
2. Click **Add Student**
3. Fill in required information:
   - First name, Last name
   - Gender
   - Date of birth
   - Level/Class
   - Admission number
4. Add parent/guardian details:
   - Parent name
   - Phone number
   - Email (optional)
5. Click **Save**

### Student Enrollment

Each student can be enrolled in:
- One level per academic year
- Multiple subjects

---

## Fee Management

### Setting Up Fee Structures

1. Go to **School** → **Fees**
2. Click **Add Fee Structure**
3. Configure:
   - Fee name (e.g., "Tuition Fee")
   - Amount (in local currency)
   - Category (Tuition, Uniform, Transport, etc.)
   - Select Level
   - Select Term
4. Click **Save**

### Viewing Fee Reports

1. Navigate to **School** → **Fees**
2. Click **View Reports**
3. Filter by:
   - Level
   - Term
   - Payment status

## Account Security

### Changing Your Password

1. Click your profile icon
2. Select **Profile**
3. Click **Change Password**
4. Enter current password
5. Enter and confirm new password
6. Click **Update**

### Session Management

- Sessions expire after 30 days of inactivity
- You can log out from any device via **Profile** → **Log Out**

## Troubleshooting

### "Cannot delete user"

- You cannot delete yourself
- You cannot delete the last owner
- Cannot delete users with active sessions

### "Email already exists"

Each user must have a unique email address. Use a different email or delete the existing account first.

### "Access denied" on a page

Check that your role has permission for that feature. Contact an admin if you need elevated access.
