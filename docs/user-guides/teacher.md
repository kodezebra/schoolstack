## Recording Exam Results

### Accessing Exam Entry

1. Go to **School** → **Exams**
2. Select the exam you need to enter grades for
3. Choose your subject
4. Enter marks for each student

### Entering Marks

**Step 1: Select Exam**

| Field | Description |
|-------|-------------|
| Exam | End of Term 1, Midterm, etc. |
| Subject | Your assigned subject |
| Level | Class you're teaching |

**Step 2: Enter Student Marks**

1. See list of students for your class
2. For each student:
   - Enter marks out of total
   - Add notes if needed (optional)
3. Save progress frequently

**Mark Entry Table:**

| Student | Admission # | Marks | /100 | Notes |
|---------|------------|-------|------|-------|
| John Doe | STU-001 | 85 | 85 | - |
| Jane Smith | STU-002 | 92 | 92 | Excellent |
| ... | ... | ... | ... | ... |

### Validation

System checks:
- Marks within valid range (0 to total)
- All students accounted for
- No duplicate entries

### Saving Grades

**Auto-Save:**
- System saves as you enter
- Look for "Saved" indicator

**Manual Save:**
- Click **Save** button
- Confirm save success

### Editing Submitted Grades

If changes needed:

1. Click on student's entry
2. Update marks
3. Add reason for change
4. Save

## Student Management

### Finding a Student

**Method 1: Search**

1. Use search bar
2. Type name or admission number
3. Select from results

**Method 2: Browse Class**

1. Select your class
2. Scroll through list
3. Click student name

### Adding Notes to Student Records

Keep track of concerns:

1. Open student profile
2. Find **Notes** section
3. Click **Add Note**
4. Enter:
   - Note type (Academic, Behavioral, Attendance)
   - Description
   - Date
5. Save

### Communication Notes

Record parent/guardian interactions:

1. Open student profile
2. Go to **Communication Log**
3. Add entry:
   - Date
   - Method (Phone, Email, Meeting)
   - Summary
   - Follow-up needed

## Common Tasks

### Entering Final Term Grades

1. Wait for exam marking complete
2. Verify all marks entered
3. Submit for approval
4. Await confirmation

### Bulk Entry Tips

For large classes:

1. Use keyboard Tab to move between fields
2. Enter marks rapidly
3. Review before final save
4. Check for typos

### Handling Absent Students

For exams:

1. Mark as "Absent" instead of 0
2. System handles accordingly
3. Notes reflect status

## Troubleshooting

### Can't find student

- Check you're viewing correct class
- Verify academic year filter
- Use exact name search

### Marks won't save

- Check internet connection
- Verify marks within range
- Try refreshing page

### Report showing outdated data

- Clear browser cache
- Check you're viewing current term
- Refresh data

### Student missing from list

- May not be enrolled in your class
- Check level assignment
- Contact administrator
