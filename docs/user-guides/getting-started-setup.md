## Step 1: Configure School Settings

Start here to set up your basic school information.

### Access Settings

1. Click **Settings** in the sidebar (or **School Settings** under Finance)
2. Or go to **School** → **Settings**

### Fill in Your School Details

| Field | What to Enter |
|-------|---------------|
| **School Name** | Your official school name |
| **Address** | Physical location |
| **Phone** | Main contact number |
| **Email** | Contact email address |

### Set Up Branding

1. **Logo Type**: Choose **Icon** (select from icons) or **Image** (upload your logo)
2. **Logo Icon**: If using icon, select one that represents your school
3. **Primary Color**: Choose your school's brand color
4. **Logo Text**: Text to show if no logo (e.g., "KZ School")

### Save Changes

Click **Save Settings** to apply your changes.

> 💡 **Tip**: These settings appear on receipts, report cards, and your public website.

## Step 2: Set Up Academic Year

Create your school's academic calendar.

### Navigate

Go to **School** → **Terms**

### Create a New Academic Year

1. Click **Add Academic Year**
2. Enter:
   - **Name**: e.g., "2025-2026" or "Term 1 2025"
   - **Start Date**: When the year begins
   - **End Date**: When the year ends
3. Toggle **Set as Current** if this is the active year
4. Click **Save**

### Add Terms

1. With the academic year selected, click **Add Term**
2. Enter:
   - **Term Name**: e.g., "Term 1", "Semester 1", or "Term 1 2025"
   - **Start Date**: First day of term
   - **End Date**: Last day of term
3. Add all terms (most schools have 3 terms)

> 💡 **Tip**: You can switch between academic years using the selector in the header.

## Step 3: Create Levels (Classes)

Define the classes or grades at your school.

### Navigate

Go to **School** → **Settings** → **Levels** section

Or go to **School** → **Settings** and scroll to **Level Management**

### Add a Level

1. Click **Add Level**
2. Enter:
   - **Name**: Full name (e.g., "Primary 1", "Nursery 2", "Senior 4")
   - **Code**: Short code (e.g., "P1", "N2", "S4")
   - **Category**: Choose:
     - **Nursery** - Early childhood
     - **Primary** - Primary school
     - **Secondary** - High school

### Common Level Structure

| Level Name | Code | Category |
|------------|------|----------|
| Pre-Nursery | PN | Nursery |
| Nursery 1 | N1 | Nursery |
| Nursery 2 | N2 | Nursery |
| Primary 1 | P1 | Primary |
| Primary 2 | P2 | Primary |
| Primary 3 | P3 | Primary |
| Primary 4 | P4 | Primary |
| Primary 5 | P5 | Primary |
| Primary 6 | P6 | Primary |
| Primary 7 | P7 | Primary |
| Senior 1 | S1 | Secondary |
| Senior 2 | S2 | Secondary |
| Senior 3 | S3 | Secondary |
| Senior 4 | S4 | Secondary |
| Senior 5 | S5 | Secondary |
| Senior 6 | S6 | Secondary |

## Step 4: Add Subjects

Set up the subjects your school teaches.

### Navigate

Go to **School** → **Subjects**

### Add Subjects

**Option A: Quick Add (Common Subjects)**

1. Click **Add Subjects**
2. Select subjects from the list:
   - Mathematics
   - English
   - Integrated Science
   - Social Studies
   - Religious Education
   - Physical Education
   - Art and Technology
   - Music, Dance and Drama
   - Local Language
   - Literacy
3. Click **Add Selected**

**Option B: Add Custom Subject**

1. In the Add form, enter:
   - **Subject Name**: e.g., "Computer Studies"
   - **Code**: e.g., "COMP" (optional)
2. Click **Add Custom**

### Assign Subjects to Levels

1. Go to **School** → **Settings**
2. Find **Level Subjects** section
3. Select a **Level**
4. Check the subjects for that level
5. Optionally assign a **Teacher** to each subject

> 💡 **Tip**: Subjects can be shared across multiple levels.

## Step 5: Add Staff Members

Add your teachers and administrators.

### Navigate

Go to **School** → **Staff**

### Add a Staff Member

1. Click **Add Staff**
2. Fill in details:

| Field | Description |
|-------|-------------|
| **First Name** | Staff member's first name |
| **Last Name** | Staff member's last name |
| **Gender** | Male or Female |
| **Email** | Work email (optional) |
| **Phone** | Contact number |
| **Role** | Teacher, Administrator, etc. |
| **Level** | Primary assigned class (optional) |

3. Upload a **Photo** (optional but recommended)
4. Click **Save**

### Assign Subjects to Teachers

1. Go to **School** → **Settings**
2. Find **Level Subjects** section
3. Select the level
4. For each subject, select the teacher from the dropdown
5. Changes auto-save

> 💡 **Tip**: Teachers can teach multiple subjects and levels.

## Step 6: Add Students

Now add your student body.

### Navigate

Go to **School** → **Students**

### Add a Single Student

1. Click **Add Student**
2. Fill in required information:

| Field | Description |
|-------|-------------|
| **First Name** | Student's first name |
| **Last Name** | Student's last name |
| **Gender** | Male or Female |
| **Date of Birth** | Use format YYYY-MM-DD |
| **Level** | Current class/enrollment |
| **Admission No** | Unique identifier (auto-generates or custom) |

3. Add **Parent/Guardian Information**:
   - Parent Name
   - Phone Number
   - Email (optional)
4. Click **Save**

### Bulk Import Students

For importing many students at once:

1. Click **Import** (button near Add Student)
2. Download the **CSV template**
3. Fill in the template:
   - firstName, lastName, gender, dob, level, admissionNo, parentName, parentPhone
4. Upload the completed CSV
5. Review the preview
6. Click **Confirm Import**

> 💡 **Tip**: Make sure the Level names match exactly what you created in Step 3.

## Step 7: Set Up Fee Structures

Define what fees students need to pay.

### Navigate

Go to **School** → **Fees**

### Create a Fee Structure

1. Click **Add Fee Structure**
2. Configure:

| Field | Description |
|-------|-------------|
| **Name** | Descriptive name (e.g., "Term 1 Tuition") |
| **Amount** | Total amount in currency |
| **Category** | Type of fee |
| **Level** | Apply to specific level or "All Levels" |
| **Term** | Select academic term |

### Fee Categories

| Category | Use For |
|----------|---------|
| **Tuition** | School fees |
| **Registration** | Enrollment fees |
| **Uniform** | Clothing items |
| **Books** | Textbooks |
| **Transport** | Bus service |
| **Boarding** | Dormitory |
| **Exam** | Examination fees |
| **Activity** | Trips, sports |
| **Other** | Miscellaneous |

> 💡 **Tip**: Create fee structures for each term so you can track payments easily.

## Step 8: Create Your First Exam

Set up an exam to start tracking student performance.

### Navigate

Go to **School** → **Exams**

### Create an Exam

1. Click **Add Exam**
2. Configure:

| Field | Description |
|-------|-------------|
| **Title** | e.g., "End of Term 1 Exam" |
| **Term** | Select the term |
| **Level** | Select class level |
| **Total Marks** | Maximum score (usually 100) |
| **Start Date** | When exam begins |
| **End Date** | When exam ends |

3. Click **Save**

### Record Results

1. Open the exam
2. Select a **Subject**
3. Enter marks for each student
4. Save

> 💡 **Tip**: Generate report cards from **School** → **Reports** once you have results.

## Troubleshooting Setup Issues

### "Level not showing in dropdown"

- Verify the level was created in Settings
- Refresh the page
- Check you're in the correct academic year

### "Cannot add student"

- Ensure level exists before adding students
- Check required fields are filled
- Verify admission number is unique

### "Fee structure not generating for students"

- Verify fee has level assigned
- Check term is selected
- Ensure academic year is active

### "Teachers not appearing in subject assignment"

- Add teacher to Staff first
- Assign level to teacher
- Check teacher's role is "teacher"
