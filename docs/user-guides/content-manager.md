**Role:** `editor` | **Version:** 1.0

## Welcome

As a Content Manager, you maintain your school's public website and handle communications. You can create and edit pages, manage the contact inbox, and keep your school's online presence current. This guide walks you through content management tasks.

## Understanding the Block Editor

SchoolStack uses a **block-based editor**. Each page is composed of blocks - individual content sections like headers, images, or feature lists.

### Block Types Available

| Block | Description |
|-------|-------------|
| **Hero** | Large banner with title, subtitle, and CTA |
| **Navbar** | Navigation menu |
| **Features** | Icon + text feature list |
| **Gallery** | Image grid |
| **Testimonials** | Customer quotes |
| **CTA** | Call-to-action section |
| **Contact** | Contact information |
| **Team** | Staff/teacher profiles |
| **FAQ** | Accordion-style questions |
| **Map** | Location embed |
| **Text** | Rich text content |
| **Image** | Single image with caption |
| **Spacer** | Empty vertical space |

## Managing Pages

### Viewing All Pages

1. Navigate to **CMS**
2. See all pages in a table:
   - Title
   - Slug (URL)
   - Status (Draft/Published)
   - Last modified

### Creating a New Page

**Step 1: Start New Page**

1. Click **New Page**
2. Enter page title
3. Auto-generates URL slug (you can edit)

**Step 2: Add Content**

1. Click **Add Block** in the editor
2. Select block type from sidebar
3. Fill in block content
4. Reorder by dragging

**Step 3: Configure Page Settings**

1. Click **Settings** (gear icon)
2. Set:
   - **Status**: Draft or Published
   - **SEO Title**: For search engines
   - **SEO Description**: For search results

**Step 4: Save**

Click **Save** to keep your changes.

### Editing an Existing Page

1. Find page in list
2. Click **Edit** (pencil icon)
3. Make changes in editor
4. Click **Save**

### Previewing Changes

Before publishing:

1. Click **Preview** button
2. View page in new tab
3. Check on mobile/tablet views
4. Return to editor if changes needed

### Publishing a Page

1. Open page in editor
2. Set status to **Published**
3. Click **Save**
4. Page is now live on your website

### Unpublishing a Page

1. Open page in editor
2. Set status to **Draft**
3. Save
4. Page removed from public site

### Deleting a Page

1. Click **⋮** menu on page
2. Select **Delete**
3. Confirm deletion

> ⚠️ **Warning:** This permanently removes the page.

## Using the Block Editor

### Adding a Hero Block

The hero is typically the first thing visitors see:

1. Click **Add Block**
2. Select **Hero**
3. Fill in:
   - **Title**: Main headline
   - **Subtitle**: Supporting text
   - **Button Text**: CTA label
   - **Button Link**: Where button goes
4. Optionally upload background image

### Adding a Features Block

Showcase key offerings:

1. Add **Features** block
2. Enter section title
3. Add feature items:
   - **Icon**: Choose from icons
   - **Title**: Feature name
   - **Description**: Brief explanation
4. Add as many features as needed

### Adding an Image

1. Add **Image** block
2. Click **Upload** or drag image
3. Optionally add caption
4. Adjust alignment (left/center/right)

### Adding a Gallery

1. Add **Gallery** block
2. Upload multiple images
3. Choose layout:
   - Grid (2-4 columns)
   - Masonry
   - Carousel
4. Set image sizes

### Adding a Contact Section

1. Add **Contact** block
2. Enter:
   - Address
   - Phone number
   - Email
   - Operating hours
3. Optionally include map

### Adding Team Members

1. Add **Team** block
2. Enter section title
3. Add members:
   - Photo
   - Name
   - Role/Title
   - Brief bio
4. Choose display style

### Reordering Blocks

1. Click and hold the block handle
2. Drag to new position
3. Drop to place

### Deleting a Block

1. Click block to select
2. Click **Delete** (trash icon)
3. Confirm

## Media Library

### Uploading Images

1. Go to **Media** (in editor or sidebar)
2. Click **Upload**
3. Select files or drag-and-drop
4. Supported: JPG, PNG, GIF, WebP
5. Max size: 5MB per image

### Using Uploaded Images

1. In any image block, click **Media**
2. Select from uploaded images
3. Click **Insert**

### Image Optimization Tips

- Resize large images before upload (max 1920px width)
- Use descriptive filenames
- Keep file sizes under 500KB when possible

## Managing Submissions

### Viewing Messages

1. Go to **Submissions**
2. See all contact form submissions:
   - **Name**: Sender
   - **Subject**: Message topic
   - **Status**: New/Read/Archived
   - **Date**: When received

### Filtering Submissions

| Filter | Use |
|--------|-----|
| **All** | See everything |
| **New** | Unread messages |
| **Read** | Already viewed |
| **Archived** | Filed messages |

### Processing a Submission

**Step 1: Open Message**

Click on submission to view full content.

**Step 2: Take Action**

- **Reply**: Click **Reply** to open email client
- **Mark as Read**: Auto-marks as read when opened
- **Archive**: Click archive to file away
- **Delete**: Remove from inbox

### Replying to Messages

1. Click **Reply** button
2. Your email client opens
3. Address is pre-filled
4. Compose and send

> 📧 **Note:** Requires email client configured on your device.

## SEO Best Practices

### Page Titles

- Keep under 60 characters
- Include target keywords
- Make it compelling for clicks

### Meta Descriptions

- Keep under 160 characters
- Summarize page content
- Include call-to-action

### URL Slugs

- Use lowercase
- Separate words with hyphens
- Include relevant keywords
- Keep short

### Image Alt Text

Always add descriptive alt text to images for accessibility and SEO.

## Troubleshooting

### Page not updating

- Clear browser cache
- Check page is published (not draft)
- Verify you're editing correct page

### Image not uploading

- Check file size (max 5MB)
- Verify format (JPG/PNG/GIF/WebP)
- Check internet connection

### Block disappeared

- Check if accidentally deleted
- Use Ctrl+Z to undo
- Reload page if needed
